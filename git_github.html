Git e GitHub



Conhecendo um pouco sobre Git e GitHub



Git


Git é uma ferramenta de acesso remoto que realiza o controle de versões, por isso é muito utilizado o termo versionamento, pois esta plataforma permite que vários Devs possam colaborar, em um mesmo projeto, ficando o log ou registro do que foi modificado, excluído ou restaudado.



GitHub


Já o GitHub pode-se dizer que é uma rede social para desenvolvedores, que nos permite guardar um projeto de forma pública ou privada, sendo o público livre para outros desenvolvedores contribuirem com com o código desenvolvido neste.
